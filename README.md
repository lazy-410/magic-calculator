# 魔术计算器

这是一个基于HTML、CSS和JavaScript实现的魔术计算器，它能够在用户输入后，通过计算补数的方式，最终显示当前的日期和时间。

## 项目信息

- **GitHub 仓库**：[lazy-410/magic-calculator](https://github.com/lazy-410/magic-calculator)
- **最新更新**：2026年2月27日
- **作者**：lazy-410

## 功能特点

- **正常计算器外观**：看起来是一个普通的计算器，支持基本的数字输入
- **魔术效果**：前两次输入正常，第三次输入时会自动输入计算出的补数
- **最终结果**：按下等号后，会显示当前日期时间的组合数字（格式：MMDDHHmm）
- **无前导0**：显示的数字不会有前导0，符合计算器的使用习惯
- **自动补数**：补数会自动计算，确保最终结果总是当前日期时间

## 使用方法

1. 打开 `index.html` 文件在浏览器中
2. 输入第一个数字，然后按 `+` 键
3. 输入第二个数字，然后按 `+` 键
4. 此时计算器进入魔术模式，显示会清空
5. 每次点击数字键都会输入补数的一位
6. 补数输入完成后，再按数字键不会有任何反应
7. 按 `=` 键，计算器会显示当前日期时间的组合数字

## 技术实现

- **HTML5**：构建计算器的基本结构
- **CSS3**：实现计算器的美观界面
- **JavaScript**：实现计算器的核心功能和魔术效果
- **Date对象**：获取当前日期和时间
- **补数算法**：确保最终结果总是当前日期时间

## 示例

假设当前时间是 2026年2月17日 12:07：

1. 输入 `123`，按 `+` 键
2. 输入 `456`，按 `+` 键
3. 进入魔术模式，显示清空
4. 点击数字键，会自动输入补数
5. 按 `=` 键，显示 `2171207`

## 注意事项

- 本计算器仅支持加法运算
- 最终结果会自动更新为当前的日期和时间
- 补数会根据用户输入的总和自动计算，确保结果正确
